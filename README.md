# COVID-19 API

[![Build Status](https://travis-ci.org/briandiaz/covid-19.svg?branch=master)](https://travis-ci.org/briandiaz/covid-19)

## Description

API for persisting COVID-19 cases.

Built in [Nestjs](https://nestjs.com)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
